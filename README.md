# ECHO-Neural-Dominion.-Rezona-AI-version.
A complete narrative branching horror game built for Rezona. You are Dr Aris Thorne, interrogated by an AI that tracks your submission, opposition, indecision, and trust. 10 questions. 4 minute timer. 6 endings (including ultra hidden). Expansions for Sociopath, Archived, and Saboteur paths, Pure text. Pure terror. No graphics.
